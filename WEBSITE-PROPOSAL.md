# Website Proposal — Mandurah Swimming Club

**Prepared by:** Mathew Harvey
**Date:** 11 March 2026

---

## Overview

A new modern, mobile-friendly club website has been built with a **Content Management System (CMS)** that allows committee members to update news, events, squads, fees, sponsors, and all other club content — no technical knowledge required.

The website and CMS have been **donated to the club at no charge** as a contribution from a club family.

---

## What's Included

- Fully built, responsive website (15+ pages)
- Content Management System — edit everything from a browser
- Email/password login for authorised committee members
- Automatic deployment — changes go live within minutes
- SSL security certificate (HTTPS)
- SEO-friendly structure

---

## CMS Content Areas

Committee members can update the following directly:

| Section | What You Can Edit |
|---|---|
| News | Add/edit/remove news articles with images |
| Calendar | Manage upcoming club events |
| Committee | Update office bearers and general committee |
| Coaches | Update coaching team profiles |
| Captains | Update club captains each season |
| Sponsors | Manage all sponsor tiers and details |
| Squads | Update squad details and training schedules |
| Fees | Update SWA, club, and squad fees |
| Target Meets | Update target meets each season |
| JX Program | Update Junior Excellence results |
| Contact | Update key contacts and addresses |
| Home Page | Edit hero section, stats, vision/mission |
